# 안드로이드 스튜디오 기초

## 알면 편한 단축키
alt + 1 : 프로젝트 툴
alt + 2 : 즐겨찾기 툴
alt + 6 : 안드로이드 모니터
alt + 7 : 스트럭쳐 툴
alt + 0 : 메세지 툴

ctr + a : 전체 선택
ctr + w : 단어 선택, 코드 블럭 전체 선택,
단어 선택 후 ctr + w : 선택 단어 인근 영역으로 선택 확장
ctr + shift + w : 선택 영역 축소

alt + 화살표 : 네비게이터 변경, 블록 이동
ctr + 화살표 : 단어 단위 커서 위치 변경, 스크롤 이동
ctr + alt + 화살표 : 네비게이션 작업 변경 (앞서 작업 하던 곳으로 이동)

ctr + h : 계층 구조
ctr + b : 해당 선언 참조

ctr + shift + backspace = 지난 편집 위치
